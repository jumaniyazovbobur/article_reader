CSS Padding


padding so'zini to'ldirish deb tarjima qilsak bo'ladi. Maqolada padding so'zini o'zini ishlatib ketamiz.


Padding alohida tomonlar uchun


Sintaksis

 padding: barchaTamonlar;

Css da elementning har bir tomoniga padding qiymat berish uchun xususiyatlar bor. 

  •  padding-top . tepa tomon padding qiymati
  •  padding-right . chap tomon padding qiymati.
  •  padding-bottom .  past tomon padding qiymati.
  •  padding-left . chap tomon padding qiymati

 

Barcha padding  xususiyatlari quyidagi qiymatlarni olishi mumkin

  • length - son korinishida qandaydir qiymat. Qiymat px,em,pt,rem va k.h lar ko'rinishida bo'lishi mumkin.
  • % -  foiz ko'rinishidagi qiymati.
  • inherit -  ( inherit nasil olish deb tarjima qilinadi). Ma'nosi shu elementning ota  elementidan  qandaydir  xususiyat qiymatini o'zlashtirib oladi. Ya'ni ota elementning padding kabi bir xil bo'ladi.

padding: 10px 50px 75px 90px;
